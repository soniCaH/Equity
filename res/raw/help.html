Texts

Click here to start the main functionality of the application

From this menu choice, you will receive the language picker (see below), after which you can navigate through all available texts, translate them and (if applicable) even use the Text-To-Speech.

Language Picker

When starting the application (with the first option - Texts (Start)), you get the option to pick your main language.

Android supports a few languages with its built-in Text-To-Speech functionality. You can pick any of these languages, or if you wish, you can select another language, which will either have translations provided inside the application, or use the Bing © Translate API to provide on-the-fly translations of the English texts (quality not controlled as it is machine-translated).
